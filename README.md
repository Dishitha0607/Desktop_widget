# 🎀 Notes & Music Player

A cute cozy desktop widget built with **React + Electron** featuring sticky notes and a pixel-style music player.

---

## ✨ Features

### 📝 Notes Widget
- Add notes
- Delete notes
- Cute cozy UI
- Responsive layout

### 🎵 Music Player
- Playlist support
- Previous / Next songs
- Play & Pause
- Real-time progress bar
- Seekable timeline
- Animated spinning vinyl
- Dynamic album covers
- Live song duration

### 🪟 Electron Desktop App
- Runs as a desktop widget
- Resizable window
- Hidden title bar aesthetic
- Floating cozy setup

## 🚀 Installation

Clone the repository:

```bash
git clone YOUR_REPO_LINK
```

Go into the project folder:

```bash
cd cozy-notes
```

Install dependencies:

```bash
npm install
```

---

## ▶ Running the App

### Start React

```bash
npm run dev
```

### Start Electron

Open another terminal:

```bash
npm run electron
```
